## 출력 형식 (반드시 준수)

### 1. 영역 개요

### 2. 레이아웃 구조 (ASCII 박스)

```
+--------------------------------------------------+
| 영역 제목                      [버튼1] [버튼2]   |
+--------------------------------------------------+
| 레이블  | [ 입력필드              ]               |
| 레이블  | ( 드롭다운 v )                          |
+--------------------------------------------------+
|  컬럼1   | 컬럼2   | 컬럼3   | 관리             |
| -------- | ------- | ------- | -------          |
|  데이터  | 데이터  | 데이터  | [수정] [삭제]    |
+--------------------------------------------------+
```

컴포넌트 표기 규칙:
- 버튼: `[버튼명]`
- 입력 필드: `[ placeholder ]`
- 드롭다운: `( 선택값 v )`
- 체크박스: `[x] 선택됨` / `[ ] 미선택`
- 라디오: `(o) 선택됨` / `( ) 미선택`
