## 기본 UI 컴포넌트

### Button
```tsx
// 사용법
<Button variant="default">저장</Button>
<Button variant="outline">취소</Button>
<Button variant="destructive" size="sm">삭제</Button>
<Button variant="ghost">닫기</Button>

// variants: default | outline | ghost | destructive | secondary | link
// sizes: default | sm | lg | icon
```

### Input / Textarea
```tsx
<Input
  placeholder="검색어를 입력하세요"
  value={value}
  onChange={(e) => setValue(e.target.value)}
/>

<Textarea
  placeholder="내용을 입력하세요"
  rows={5}
  value={value}
  onChange={(e) => setValue(e.target.value)}
/>
```

### Select (드롭다운)
```tsx
<Select value={value} onValueChange={setValue}>
  <SelectTrigger>
    <SelectValue placeholder="선택하세요" />
  </SelectTrigger>
  <SelectContent>
    <SelectItem value="option1">옵션 1</SelectItem>
    <SelectItem value="option2">옵션 2</SelectItem>
  </SelectContent>
</Select>
```

### Dialog (모달)
```tsx
<Dialog open={open} onOpenChange={setOpen}>
  <DialogTrigger asChild>
    <Button>열기</Button>
  </DialogTrigger>
  <DialogContent>
    <DialogHeader>
      <DialogTitle>제목</DialogTitle>
    </DialogHeader>
    {/* 내용 */}
    <DialogFooter>
      <Button variant="outline" onClick={() => setOpen(false)}>취소</Button>
      <Button onClick={handleSave}>저장</Button>
    </DialogFooter>
  </DialogContent>
</Dialog>
```

### Badge (상태 표시)
```tsx
<Badge variant="default">활성</Badge>
<Badge variant="secondary">대기</Badge>
<Badge variant="destructive">오류</Badge>
<Badge variant="outline">완료</Badge>
```

## DataGrid (TanStack Table)

### 기본 구조
```tsx
import {
  useReactTable,
  getCoreRowModel,
  flexRender,
  ColumnDef,
} from "@tanstack/react-table";

const columns: ColumnDef<RowType>[] = [
  {
    accessorKey: "systemId",
    header: "ID",
    size: 100,
    cell: ({ row }) => (
      <span className="font-mono text-xs text-primary">
        {row.original.systemId}
      </span>
    ),
  },
  {
    accessorKey: "name",
    header: "이름",
    size: 200,
  },
  // ... 더 많은 컬럼
];

const table = useReactTable({
  data: rows,
  columns,
  getCoreRowModel: getCoreRowModel(),
});
```

### 컬럼 너비 주의
- `table-fixed` 사용 시 `size` 값은 **비율** (절대 px 아님)
- 좁은 컬럼은 작게, 넓은 컬럼은 크게 상대적으로 설정

## 마크다운 렌더링

Tailwind CSS v4에서는 `@tailwindcss/typography` 미설치 → `prose` 클래스 불가.
반드시 `markdown-body` 클래스 사용.

```tsx
// ❌ 잘못된 방법
<div className="prose prose-sm max-w-none">{/* ... */}</div>

// ✅ 올바른 방법
<div className="markdown-body text-sm"
  dangerouslySetInnerHTML={{ __html: renderedHtml }}
/>
```

`markdown-body` CSS는 `globals.css`에 정의되어 있음.

## 공통 패턴

### 로딩 상태
```tsx
{isLoading ? (
  <div className="flex items-center justify-center h-32">
    <Loader2 className="w-6 h-6 animate-spin text-muted-foreground" />
  </div>
) : (
  <DataTable data={rows} columns={columns} />
)}
```

### 빈 상태
```tsx
{rows.length === 0 && (
  <div className="text-center py-12 text-muted-foreground">
    <FileX className="w-12 h-12 mx-auto mb-3 opacity-30" />
    <p>데이터가 없습니다.</p>
  </div>
)}
```

### 에러 상태
```tsx
{error && (
  <div className="p-4 bg-destructive/10 text-destructive rounded-md text-sm">
    오류가 발생했습니다: {error.message}
  </div>
)}
```

### 토스트 알림
```tsx
import { toast } from "sonner";  // 또는 프로젝트 선택 라이브러리

toast.success("저장되었습니다.");

```

## 파일 다운로드 패턴

```ts
// BOM 포함 UTF-8 마크다운 다운로드
const blob = new Blob(["\uFEFF" + markdownContent], {
  type: "text/markdown;charset=utf-8"
});
const url = URL.createObjectURL(blob);
const a = document.createElement("a");
a.href = url;
a.download = `파일명_${new Date().toISOString().slice(0, 10)}.md`;
a.click();
URL.revokeObjectURL(url);
```
