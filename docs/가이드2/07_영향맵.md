## 순서(sortOrder) 관련 기능 구현 시

- 드래그앤드롭 후 즉시 저장 or 명시적 저장 버튼 결정
- 신규 생성 시 max(sortOrder) + 1 로직 포함
- 조회 시 ORDER BY sort_order 포함 확인

## 명세 필드(spec, content 등) 저장 시

- saveContentVersion() 호출로 이전 값 이력 적재 필요
- 이력 적재 대상 필드 목록: 각 엔티티의 주요 명세 필드

## 프로젝트 멤버 관련 기능 구현 시
