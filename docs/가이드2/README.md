## 핵심 원칙 (변경 금지)

- **논리 삭제** 원칙 — 데이터는 물리 삭제하지 않는다
- **systemId** 채번 방식 — DB sequence 테이블 기반 (T-00001 형식)
- **AI 태스크** 비동기 큐 방식 — NONE → RUNNING → SUCCESS/FAILED
- **계층 구조** — 과업 → 요구사항 → 사용자스토리 / 단위업무 → 화면 → 영역 → 기능
